# QKD Forensic Dataset

This repository contains a realistic and physics-informed Quantum Key Distribution (QKD) forensic dataset.

- **qkd_timeseries_lstm_v2.csv**

- intercept_resend
- detector_blind
- fiber_tap

## Features
- QBER  
- Photon Count  
- Latency  
- Abort Flag
