How to use this folder with your GitHub Pages site (MadhabCD.github.io):

1. Download this ZIP and extract it.
2. Inside you will see a folder named 'qkd-dataset' with an index.html inside.
3. Go to https://github.com/MadhabCD/MadhabCD.github.io
4. Click 'Add file' -> 'Upload files'.
5. Drag and drop the 'qkd-dataset' folder into the repo root.
   After commit, your structure will include:
      /qkd-dataset/index.html

6. Your dataset page will then be available at:
      https://madhabcd.github.io/qkd-dataset/

7. (Optional) Add a link in your main navigation menu:
   For example, in your main index.html navbar:

      <li><a href="/qkd-dataset/">QKD Dataset</a></li>

That's it! Your personal website will now have a separate QKD Dataset page.
