QF-LOG: Quantum Forensic Dataset for QKD Networks

By Madhab Chandra Das · Quantum Forensics Framework

Overview

QF-LOG is a physics-grounded synthetic dataset for forensic analysis of Quantum Key Distribution (QKD) networks. It captures realistic behaviour of four scenarios: normal, intercept_resend, detector_blind, and fiber_tap.

50,000 samples (tabular) + time-series LSTM version 4 attack classes QBER · photon_count · latency · abort_flag

Files

  • full_qkd_dataset_realistic_50000.csv – main tabular dataset
  • qkd_timeseries_lstm_v2.csv – version 2, sequence-format for LSTM/1D-CNN
  • dataset_card.md – dataset card (HuggingFace-style)
  • README.md – repository level documentation
  • supplementary_dataset_description.pdf – PDF description for papers

Feature Schema

qber           float   Quantum Bit Error Rate per frame
photon_count   int     Photon detection count
latency_ms     float   Classical processing latency
abort_flag     int     0 = OK, 1 = Abort
label          string  normal / intercept_resend / detector_blind / fiber_tap
session_id     (v2)    integer time-series session index
frame_index    (v2)    integer frame position in session
